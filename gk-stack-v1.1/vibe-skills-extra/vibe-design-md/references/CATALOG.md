## How to use this catalog

1. Find the site by name or category
2. Copy the exact slug from the Slug column
3. Fetch URL: `[Base URL][slug]/DESIGN.md`

If a site isn't listed — use Mode B (URL generation).

## Sites not in the catalog

Request a DESIGN.md generation for any site not listed above using Mode B:
```
design-md: https://[site-url]
```

The agent will fetch the site, extract design tokens from CSS,
and generate a complete DESIGN.md in the same 9-section format.

Common sites not yet in the catalog (use Mode B):
- Framer, Webflow Studio, Figma Community
- GitHub, GitLab, Bitbucket
- Anthropic.com (different from Claude UI)
- Y Combinator, Product Hunt
- Any client's competitor site
- Any site you want to reference
