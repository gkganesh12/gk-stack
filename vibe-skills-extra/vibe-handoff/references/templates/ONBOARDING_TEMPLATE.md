## Setup — step by step

Run these commands in order. Each one should complete without errors.
If something fails, check GOTCHAS.md before debugging.

```bash
# 1. Clone the repository
git clone [repo URL]
cd [project name]

# 2. Install dependencies
npm install                    # frontend
pip install -r requirements.txt --break-system-packages  # backend (if Python)

# 3. Set up environment variables
cp .env.example .env
# Open .env and fill in values from CREDENTIALS.md

# 4. Set up the database
npx prisma migrate dev         # runs all migrations
npx prisma db seed             # seeds initial data (if applicable)

# 5. Start the development server
npm run dev
# App should be running at http://localhost:3000
```

**Verify setup is working:**
- [ ] App loads at http://localhost:3000 without errors
- [ ] Can log in with test account: [test credentials — not real ones]
- [ ] No red errors in browser console on page load

## Running tests

```bash
npm test                 # runs all tests
npm test -- --watch      # watch mode during development
npm run test:e2e         # end-to-end tests (requires running server)
```

All tests should pass on a clean setup. If any fail, check GOTCHAS.md first.
