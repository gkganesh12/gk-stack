## Quality standards for generated DESIGN.md files

**Every generated file must have:**
- Minimum 6 colour tokens with exact hex values
- Complete typography table with all 10 roles filled
- Button component with all 5 states (default, hover, active, disabled, focus)
- At least one card and one input component
- Specific do's and don'ts (not generic)
- At least 3 agent prompts

**Mark inferred values with `*`:**
Any value that couldn't be extracted from CSS and was inferred from visual inspection
gets an asterisk: `#5E6AD2*` — and a note in the header.

**Flag JS-rendered gaps:**
If the site is a SPA and computed styles couldn't be extracted:
Add a warning block at the top of the file listing which sections have incomplete data.
